  • Click the lines \ rows or any cells to highlight the range for changes.
  • To manually set the range, here's the example: M4-M34 : 100
    • Where (1) The M4-M34 stands for the specific range you are targeting for the change of the automated value changed
    • and (2) after the semi-column, the new value to assign to the range, i.e., 100, is to be set.
  • To set a new date value, follow this example: M4-M34 : new Date('Mon Mar 01 2016 10:00:00')
    • Where: (3) The M4-M34 stands for the specific range you are targeting for the change of the automated value changed;
    • and (4) after the semi-column, the new Date value is set to assign to the given range, i.e., new Date('Mon Mar 01 2016 10:00:00').
